To build:

$ docker build -t bagapp:dev .

To run:

$ docker run \
    -it \
    -d \
    --rm \
    --name bagappcontainer \
    -v ${PWD}:/app \
    -v /app/node_modules \
    -p 3001:3000 \
    -e CHOKIDAR_USEPOLLING=true \
    bagapp:dev
